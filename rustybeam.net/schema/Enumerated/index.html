Enumerated Schema

The enumerated datatype is a type that allows for a fixed set of values, known as enumerators. Each enumerator is a specific value that can be used. The value is valid when it matches one of the enumerators defined in the schema.

Schema Definition

Schema URL: https://rustybeam.net/schema/Enumerated

Properties

Property Type Cardinality Description
enumerator Text 1..n A specific value that can be used for this enumerated type
