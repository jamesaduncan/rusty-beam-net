HostConfig Schema

The HostConfig schema defines the configuration for a virtual host in Rusty Beam. Each host can have its own root directory, custom server header, and plugin pipeline.

Schema Definition

Schema URL: http://rustybeam.net/schema/HostConfig

Properties

Property Type Cardinality Description
hostname Text 1..n The hostname for virtual host routing (e.g., "localhost", "example.com"). There can be more than one, and each will use the same config
hostRoot Text 1 The root directory for files served by this virtual host
serverHeader Text 0..1 Custom Server header value for HTTP responses from this host
plugin Plugin 0..n Plugin configurations that form the processing pipeline for this host
