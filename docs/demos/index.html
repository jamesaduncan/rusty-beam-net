Rusty Beam Demos

These demos showcase the capabilities of Rusty Beam's plugin architecture and selector-based content manipulation.

Guestbook

A real-time collaborative guestbook demonstrating:

  • Selector-based content updates
  • Role-based authorization (administrators can delete entries)
  • DOM-aware streaming for real-time updates
  • OAuth2 authentication integration

Todo List

A simple todo list application demonstrating:

  • CSS selector-based CRUD operations
  • Authentication-based permissions (only logged-in users can delete todos)
  • Minimal styling for easy code comprehension
  • Real-time updates with DOM-aware streaming
